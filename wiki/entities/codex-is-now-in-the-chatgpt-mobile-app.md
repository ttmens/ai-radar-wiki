# Codex is now in the ChatGPT mobile app

## 中文摘要
OpenAI 将 Codex（代码生成模型）集成到 ChatGPT 移动应用中，用户现在可以在手机端通过对话直接获取代码建议、解释和调试帮助。这一举措显著提升了 ChatGPT 在移动场景下的实用性，尤其是对开发者用户而言，无需打开桌面 IDE 即可快速编码。商业上，它增强了 ChatGPT 的差异化竞争力，吸引专业用户并推动订阅转化。产品创新在于将代码助手无缝融入移动聊天界面，降低使用门槛，扩大 AI 编程的普及度。
