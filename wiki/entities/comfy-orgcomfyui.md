# Comfy-Org/ComfyUI

## 概览
The most powerful and modular diffusion model GUI, api and backend with a graph/nodes interface.

## 中文摘要
ComfyUI是基于节点图架构的开源扩散模型平台，集成模块化GUI、API与后端。其核心创新在于将AI生成流程可视化与可编程化，支持拖拽节点自由编排模型与处理逻辑。对AI产品经理而言，该工具大幅降低复杂工作流的开发门槛，依托活跃的第三方插件生态，为快速验证生成式应用原型、构建企业级定制流水线及探索SaaS商业化提供了底层基础设施支撑。
