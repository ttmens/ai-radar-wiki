# Zerostack – A Unix-inspired coding agent written in pure Rust

## 中文摘要
Zerostack 是一个受 Unix 哲学启发的 AI 编码代理，完全用 Rust 实现。它借鉴了 Unix 的模块化、管道和组合原则，旨在提供高效、可扩展的代码生成与执行能力。使用 Rust 带来了内存安全和极致性能，适合构建可靠的开发者工具。产品创新点在于将经典操作系统设计理念融入现代 AI 代理，有望提升代码生成的透明度和可控性。商业上可定位为面向开发者的高效辅助工具，尤其适合对性能和安全性有高要求的场景。
