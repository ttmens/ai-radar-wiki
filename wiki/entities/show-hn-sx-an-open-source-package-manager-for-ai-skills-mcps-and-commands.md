# Show HN: Sx – an open-source package manager for AI skills, MCPs, and commands

## 中文摘要
Sx 是一个开源包管理器，专为 AI 技能、MCP（模型上下文协议）和命令设计，类似于 npm 但面向 AI 生态。它允许开发者发布、发现和组合可复用的 AI 模块，降低构建复杂 AI 工作流的门槛。该项目通过标准化技能和工具集成，推动 AI 应用的模块化和生态化，对产品经理而言，意味着更快的原型搭建和更丰富的功能组合可能性。其开源特性也鼓励社区贡献，有望形成类似 JavaScript 生态的 AI 技能市场，提升团队协作效率。
