# Show HN: InsForge – Open-source Heroku for coding agents

## 中文摘要
InsForge 是一个开源平台，定位为‘编码代理的 Heroku’，旨在简化 AI 编码代理的部署、托管和运维。它提供类似 Heroku 的 PaaS 体验，让开发者能够快速将代码代理（如基于 LLM 的自动化编程工具）部署到云端，并自动处理扩展、监控和基础设施管理。该平台强调开源和可自托管，降低了对云服务商的依赖，同时提升了编码代理在生产环境中的易用性和可扩展性。对于 AI 产品经理而言，InsForge 填补了编码代理从实验到规模化落地的工具链空白，有望推动更多 AI 驱动开发工具进入实际业务场景。
