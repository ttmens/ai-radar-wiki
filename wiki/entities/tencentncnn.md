# Tencent/ncnn

## 概览
ncnn is a high-performance neural network inference framework optimized for the mobile platform

## 中文摘要
ncnn是腾讯开源的高性能移动端神经网络推理框架，专为手机等边缘设备优化。其核心优势在于极致轻量化与跨平台兼容，支持零依赖部署，大幅降低移动端算力与内存开销。对AI产品经理而言，它是实现端侧AI落地的关键基建，能有效提升应用响应速度、保障用户隐私，并显著削减云端算力成本，为打造低延迟智能产品提供核心支撑。
