# infiniflow/ragflow

## 概览
RAGFlow is a leading open-source Retrieval-Augmented Generation (RAG) engine that fuses cutting-edge RAG with Agent capabilities to create a superior context layer for LLMs

## 中文摘要
RAGFlow是领先的开源RAG引擎，深度融合检索增强生成与Agent能力，为LLM构建高质量上下文层。其创新在于提供深度文档解析、可视化工作流编排及精准召回控制，显著降低企业构建知识库应用的门槛。对AI产品经理而言，该工具可大幅缩短产品落地周期，支持灵活定制业务逻辑，是打造高准确率企业级AI应用的核心基础设施，具备极高的工程复用价值。
