# milvus-io/milvus

## 概览
Milvus is a high-performance, cloud-native vector database built for scalable vector ANN search

## 中文摘要
Milvus是一款高性能云原生向量数据库，专为大规模近似最近邻搜索设计。对AI产品经理而言，它是构建RAG、语义检索等LLM应用的核心基础设施。其分布式架构支持弹性扩缩容与高并发查询，有效降低海量向量数据的存储与检索成本。产品通过标准化API封装复杂计算，提供开箱即用的检索能力，助力团队快速落地AI场景，大幅缩短商业化周期。
