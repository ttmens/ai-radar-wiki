# Blaise – A modern self-hosting zero-legacy Object Pascal compiler targeting QBE

## 中文摘要
Blaise是一款面向Object Pascal的现代化自托管编译器，采用“零历史包袱”架构并直连轻量级QBE后端。其核心价值在于摆脱传统技术债，实现高效跨平台代码生成。对AI产品经理而言，该工具虽非AI模型，但为边缘计算与嵌入式AI提供了轻量级系统语言支持，可优化特定硬件的推理部署链路，降低底层工具链维护成本，具备垂直生态整合潜力。
