# Gmail registration now requires scanning a QR code and sending a text message

## 中文摘要
谷歌近期调整Gmail注册流程，强制引入二维码扫描与短信验证双重机制。该举措旨在通过增加交互摩擦有效拦截自动化脚本与黑产批量注册，全面升级账号安全基线。对AI产品经理而言，平台级反爬与风控策略升级将直接影响依赖Gmail进行OAuth授权或数据抓取的AI应用集成方案。团队需提前适配多因子认证链路，并在保障安全的前提下优化新用户引导体验，精准平衡风控强度与转化漏斗。
