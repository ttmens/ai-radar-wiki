# Gemini API File Search is now multimodal

## 中文摘要
谷歌Gemini API的File Search功能现已支持多模态检索，开发者可直接对图像、音频、视频及文档进行统一索引与语义搜索。该更新大幅降低了构建跨媒体RAG应用的技术门槛，无需额外部署多模态解析管线。对AI产品经理而言，这意味着可快速打造支持富媒体知识库的智能助手，显著提升信息召回率与用户体验，加速多模态AI产品的商业化落地。
