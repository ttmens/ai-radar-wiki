# Anthropic has acquired the dev tools startup used by OpenAI, Google, and Cloudflare

## 中文摘要
Anthropic 收购了纽约初创公司 Stainless（2022年成立），该公司专注于自动化 SDK（软件开发工具包）的创建与维护，其客户包括 OpenAI、Google 和 Cloudflare。通过将 Stainless 的自动化工具整合，Anthropic 旨在降低开发者调用其 API 的集成门槛，提升开发者体验，并巩固其在 AI 生态中的基础设施地位。此举不仅强化了工具链，也可能影响 API 设计范式，推动更标准化的 SDK 输出模式。对 AI 产品经理而言，这标志着头部模型公司从提供模型向提供完整开发生态延伸，商业价值在于锁定开发者群体并降低迁移成本。

## 新闻摘要
Stainless, a New York-based startup, founded in 2022, rose to prominence in the emerging AI industry for automating the creation and maintenance of software development kits, or SDKs — the libraries developers use to interact with APIs.
