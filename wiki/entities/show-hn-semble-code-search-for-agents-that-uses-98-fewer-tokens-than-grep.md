# Show HN: Semble – Code search for agents that uses 98% fewer tokens than grep

## 中文摘要
Semble 是一个专为 AI 代理设计的代码搜索工具，相比传统 grep 方式，其 token 消耗降低了 98%。该产品通过优化检索算法，使代理在搜索和理解代码库时大幅减少 LLM 调用的 token 开销，从而显著降低推理成本并提升响应速度。对于构建代码生成、调试助手或自主编程代理的产品团队，Semble 提供了一种高效、经济的基础设施方案，其核心创新在于将代码搜索的效率和准确性针对代理工作流进行了重新设计。
