# daytonaio/daytona

## 概览
Daytona is a Secure and Elastic Infrastructure for Running AI-Generated Code

## 中文摘要
Daytona是专为运行AI生成代码设计的安全弹性基础设施。通过动态沙箱隔离与按需扩缩容，有效解决AI代码执行的安全信任与资源调度难题。对AI产品经理而言，它提供标准化运行时环境，可无缝集成至AI编程助手或智能体工作流，大幅降低沙箱运维成本与执行风险，加速AI开发工具商业化落地，是构建可信AI平台的关键底层组件。
