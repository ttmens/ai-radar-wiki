# marimo-team/marimo

## 概览
A reactive notebook for Python — run reproducible experiments, query with SQL, execute as a script, deploy as an app, and version with git. Stored as pure Python. All in a modern, AI-native editor.

## 中文摘要
Marimo是一款面向Python的响应式AI原生笔记本。核心创新在于将文件存为纯Python代码，彻底解决传统Notebook的执行状态混乱与Git协作难题，保障实验可复现。它支持SQL查询、脚本运行与一键部署，无缝打通从数据探索到应用交付的全链路。内置AI编辑器大幅提升编码效率，为AI团队提供低摩擦、高协作的研发工作流，有效降低模型落地成本并加速产品迭代。
