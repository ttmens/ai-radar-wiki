# We stopped AI bot spam in our GitHub repo using Git's –author flag

## 中文摘要
本文介绍了一种利用Git原生--author参数拦截AI机器人垃圾提交的高效方案。该方案无需复杂中间件，仅通过校验提交者元数据即可精准过滤自动化低质代码，显著降低仓库维护成本。对AI产品经理而言，此实践揭示了在AI自动化泛滥背景下，利用底层协议元数据进行轻量级治理的可行性，为构建防刷机制与维护开发者生态健康提供了低成本、高兼容性的产品防御思路。
