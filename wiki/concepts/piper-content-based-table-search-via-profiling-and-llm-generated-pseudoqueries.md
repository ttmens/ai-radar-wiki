# PIPER: Content-Based Table Search via profiling and LLM-Generated Pseudoqueries

## 中文摘要
PIPER 是一种基于表格内容（而非仅元数据）的搜索方法，通过数据画像（profiling）和 LLM 生成的伪查询来提升数据湖/数据空间中表格的发现效率。传统方法依赖缺失或不准确的元数据，而 PIPER 利用 LLM 自动生成模拟用户意图的查询，结合表格的列类型、值分布等特征进行匹配。该方法可显著降低数据检索门槛，助力企业快速复用和分析海量表格数据，尤其适用于数据目录、数据市场等产品场景。
