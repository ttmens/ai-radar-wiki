# RGB-only Active 3D Scene Graph Generation for Indoor Mobile Robots

## 中文摘要
本论文提出仅依赖RGB摄像头的主动式3D场景图生成方法，无需LiDAR或RGB-D深度传感器，即可为室内移动机器人构建度量3D重建和语义场景图。该技术突破了专用深度传感器的硬件限制，降低了机器人部署成本和门槛，使普通RGB相机（如固定监控摄像头）也能实现场景理解。核心创新在于融合主动探索策略与单目深度估计，实时生成包含物体、空间关系和拓扑结构的动态场景图。对产品经理而言，这意味着低成本机器人、消费级设备可具备高级环境感知能力，拓展了家庭服务、仓储巡检等场景的落地可能性。
