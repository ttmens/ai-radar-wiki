# SPATIOROUTE: Dynamic Prompt Routing for Zero-Shot Spatial Reasoning

## 中文摘要
本研究提出SPATIOROUTE，一种动态提示路由方法，用于零样本空间推理。该方法通过自动选择最合适的提示策略，引导视觉语言模型（VLM）对自我中心视频中的3D物体位置、场景属性和方向关系进行推理。技术核心在于无需微调即可适应多种空间查询，降低了模型应用成本。商业价值体现在对AR/VR辅助、机器人导航、智能视频分析等场景的快速部署。产品创新在于将空间推理能力从静态图像扩展到动态视频，且无需标注数据。
