# lucidrains/vit-pytorch

## 概览
Implementation of Vision Transformer, a simple way to achieve SOTA in vision classification with only a single transformer encoder, in Pytorch

## 中文摘要
该项目提供了Vision Transformer的PyTorch实现，将Transformer架构成功迁移至视觉分类。仅凭单一编码器即达SOTA精度，大幅简化传统CNN结构。对AI产品经理而言，其商业价值在于统一多模态底层技术栈，显著降低算力与部署成本，加速视觉产品迭代；同时为图像识别、内容审核及多模态应用提供高扩展性基座，是构建下一代视觉AI产品的核心能力组件。
