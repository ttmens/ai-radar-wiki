# screenpipe/screenpipe

## 概览
Run agents that work based on what you do. 24/7 local screen & mic recording for the superintelligence era

## 中文摘要
该项目提供本地化、全天候的屏幕与麦克风录制能力，为AI Agent构建持续的个人行为上下文数据流。技术核心在于端侧多模态采集与隐私保护。产品创新在于将被动记录转化为Agent主动感知的基础设施，使AI能基于真实操作流提供自动化服务。对PM而言，它是打造个性化助手与记忆型Agent的关键底层能力，兼具高商业潜力与数据合规优势。
