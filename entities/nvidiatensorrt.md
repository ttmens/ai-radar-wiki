# NVIDIA/TensorRT

## 概览
NVIDIA® TensorRT™ is an SDK for high-performance deep learning inference on NVIDIA GPUs. This repository contains the open source components of TensorRT.

## 中文摘要
TensorRT是NVIDIA专为GPU深度学习推理打造的高性能SDK，核心聚焦模型压缩、算子融合与精度校准等底层优化技术。对AI产品经理而言，其商业价值在于显著降低推理延迟与云端算力成本，提升产品并发处理能力与用户体验。开源组件的开放进一步降低了部署门槛，加速了从算法到生产环境的标准化落地，是构建高性能AI应用不可或缺的底层基础设施。
