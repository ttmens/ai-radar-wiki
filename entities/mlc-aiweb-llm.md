# mlc-ai/web-llm

## 概览
High-performance In-browser LLM Inference Engine 

## 中文摘要
该项目是基于WebGPU的高性能浏览器端大模型推理引擎，支持主流LLM在客户端直接运行。技术核心为模型量化编译与Web原生加速，实现零服务器依赖与数据本地化。对AI产品经理而言，其商业价值在于大幅削减云端算力成本，满足隐私合规要求，并赋能离线AI助手、隐私优先型SaaS及轻量化Web应用等创新形态，是端侧AI产品落地的关键基础设施。
