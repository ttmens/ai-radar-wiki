# rtk-ai/rtk

## 概览
CLI proxy that reduces LLM token consumption by 60-90% on common dev commands. Single Rust binary, zero dependencies

## 中文摘要
该项目是一款基于Rust的轻量级CLI代理，专为开发者终端场景设计。其核心通过智能请求路由与上下文优化，将常见开发指令的LLM Token消耗降低60%-90%。作为零依赖单二进制文件，部署极简。对AI产品经理而言，它直击大模型应用的高成本痛点，为AI编程助手与终端智能体提供底层降本方案，具备显著的商业价值与开发者工具集成潜力。
